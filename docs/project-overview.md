## **About the Project**
This is a website project that was built for Close AI, a real estate lead generation agency helping real estate agents generate seller leads. 

What makes this company unique is that they ONLY focus on generaing seller leads and they do this with a unique business model, a pay-per-lead model meaning that agents do not have to spend thousands upfront, they only pay per lead. This saves them so much money and its risk free. There are no contracts or any commitments, users can cancel anytime.

It is imperatve to hyper focus on seller leads and the benefits if the business model as this is what will rank on search engines. For more info review the website content to get a complete picture of what the company does: content/main, content/pages

Tech Stack Used:
* **Next.js (App Router)** + **TypeScript** + **TailwindCSS (latest v4)**
* **Shadcn/UI (latest version)**
* **ESLint** configured
* **Turbopack** enabled for fast dev builds
